# ✨ HabitRabbit – Smart Habit Tracker App 🐰

A beautiful and interactive Habit Tracker web application designed to help users build better daily routines, stay productive, and track progress visually.

Built with ❤️ using HTML, CSS, JavaScript, and Chart.js.

# 📸 Key Functionalities

### ✔ Complete / Undo Habits
Users can easily mark habits as completed and undo them anytime.

### ✏ Edit Habits
Update habit names instantly with the edit feature.

### 🗑 Delete Habits
Remove unwanted habits from the tracker.

### 📈 Analytics Dashboard
Visualize habit progress using charts and statistics.

### 🌙 Dark Mode
Switch between light and dark themes for better user experience.

---

# 💾 Data Persistence

All data is stored using browser LocalStorage.

This includes:
- Habits
- Completion status
- Streak count
- Theme preferences

Your data remains saved even after refreshing the browser.
